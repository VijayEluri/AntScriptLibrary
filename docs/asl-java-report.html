Ant Script Library: asl-java-report.xml

Overview

asl-java-report.xml contains properties and targets for various generated reports about Java code.

Properties

All properties defined with this module are prefixed with "java-report.".

Name Default Value Description
java-report.report-dir ${common.dist-dir}/reports The top level directory under which all generated reports will be placed.
java-report.coverage-dir ${java-report.report-dir}/coverage The directory where the Cobertura test coverage report will be generated.
java-report.javadoc-dir ${java-report.report-dir}/javadoc The directory where the Javadoc report will be generated.
java-report.jdepend-data-dir ${common.target-dir}/jdepend-data There directory where the JDepend analysis data will be stored.
java-report.jdepend-dir ${java-report.report-dir}/jdepend The directory where the JDepend report will be generated.
java-report.pmd-dir ${java-report.report-dir}/pmd The directory where the PMD code quality report will be generated.
java-report.test-dir ${java-report.report-dir}/test The directory where the JUnit test report will be generated.

Targets

report-all

Generates all available reports.

report-checkstyle

Generates a Checkstyle report.

The location of this report is controlled by the property java-report.checkstyle-dir.

report-cobertura

Generates a Cobertura test coverage report.

The location of this report is controlled by the property java-report.coverage-dir.

report-cpd

Generates a Copy-Paste Duplication (TODO: check link) report.

The location of this report is controlled by the property java-report.cpd-dir.

report-findbugs

Generates a FindBugs report.

The location of this report is controlled by the property java-report.findbugs-dir.

report-javadoc

Generates JavaDoc (TODO: check link) for all the source code underneath the directory specified by the property java-build.src-dir.

The location of this report is controlled by the property java-report.javadoc-dir.

report-jdepend

Generates a JDepend (TODO: check link) report.

The location of this report is controlled by the property java-report.jdepend-dir.

report-pmd

Generates a PMD code quality report.

The location of this report is controlled by the property java-report.pmd-dir.

report-tests

Generates a test report from the set of JUnit xml reports found in the directory ${java-test.test-data-dir}. Note that both the test-unit target and the test-integration target will place their data files in that directory.

The location of this report is controlled by the property java-report.test-dir.
